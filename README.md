勤務時間計算機
勤怠時間の「10進法表記」を自動計算するWebアプリケーションです。 
面倒な時間の引き算や、分→時間の変換作業を効率化します。

-主な機能
自動計算: 出社・退社時間と休憩時間を入力するだけで、実労働時間を即座に算出。
10進法変換: 「3時間30分」を「3.50時間」のように、給与システムでよく使われる形式に自動変換します（15分単位）。
休憩時間の柔軟な対応: 休憩は「追加」ボタンでいくつでも増やせるため、昼休憩＋小休憩のような不規則なパターンにも対応可能です。

インストール不要: ブラウザがあればどこでも動きます。

-使い方
出社時間と退社時間を選択します（15分単位）。
休憩時間（開始・終了）を入力します。休憩が複数ある場合は「＋追加」ボタンで枠を増やせます。
画面下部に実労働時間（時間と分、および10進数）が自動的に表示されます。
